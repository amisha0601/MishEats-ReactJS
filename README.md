A frontend food ordering application built while learning React fundamentals and modern frontend concepts.

The project focuses on reusable components, routing, state management, custom hooks, testing, and responsive UI design.

## ✨ Features

- Restaurant listing page
- Restaurant search functionality
- Top-rated restaurant filter
- Dynamic restaurant menu page
- Accordion-based menu categories
- Add / Remove items from cart
- Quantity update controls
- Cart total calculation
- Toast notifications
- Offline detection support
- Interactive memory game in offline mode
- Fully responsive design
- Unit and integration testing

---

## 🛠 Tech Stack

| Category | Technologies |
|-----------|-------------|
| Frontend | React 19 |
| State Management | Redux Toolkit, React Redux |
| Styling | Tailwind CSS |
| Routing | React Router DOM |
| Build Tool | Parcel |
| Notifications | React Hot Toast |
| Testing | Jest, React Testing Library |

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/amisha0601/MishEats-ReactJS.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

Run tests:

```bash
npm test
```

Build production version:

```bash
npm run build
```
